# Sistema Mantenedor de Procesos
**Estudiante:** Edison Tigrero Camacho /George Quintero Nazareno
**Materia:** Ingeniería de Software / Sistemas  

## Descripción
Este proyecto consiste en un script de mantenimiento automatizado para la gestión de recursos del sistema. El proceso se encarga de optimizar el almacenamiento eliminando archivos temporales de paquetes (`yum`) y dependencias locales de desarrollo (`node_modules`).

## Arquitectura
- **Lenguaje:** Bash Scripting
- **Entorno:** Windows / Visual Studio Code
- **Control de Versiones:** Git / GitHub

## Instrucciones de Uso
1. Dar permisos de ejecución: `chmod +x limpieza.sh`
2. Ejecutar el proceso: `./limpieza.sh`
